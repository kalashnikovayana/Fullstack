Властивість display та box-sizing

← На головну

1. display: block

Елемент займає весь рядок. Ширина = 100% батька. Наступний елемент починається з нового рядка.
Я span, але display: block — займаю весь рядок І я теж займаю весь рядок

2. display: inline

Елемент займає лише свій вміст. Ширина і висота через CSS не діють. Елементи стоять в одному рядку.

Інлайн 1 Інлайн 2 Інлайн 3

3. display: inline-block

Золота середина: елемент стоїть в рядку (як inline), але реагує на ширину, висоту, вертикальні відступи (як block). Зручно для кнопок, карток, навігаційних пунктів.
Картка 1 Картка 2 Картка 3

4. display: flex

Контейнер стає flex-контейнером: діти розташовуються в ряд (або колонку), можна гнучко управляти відступами, вирівнюванням і розподілом простору.

5. display: none

Елемент повністю прихований і не займає місця на сторінці. Нижче є елемент, але ти його не бачиш:

Цей текст прихований через display: none — він є в HTML, але не видно.

Після прихованого параграфа — одразу цей текст.

6. box-sizing

box-sizing: content-box (дефолт) — ширина width означає тільки вміст. Padding та border додаються зверху. Тому реальна ширина = width + padding + border.

box-sizing: border-box — ширина width включає і padding, і border. Реальна ширина = width (не більше). Саме цей варіант рекомендується і зазвичай задається через * { box-sizing: border-box; }.

width: 200px
padding: 20px
border: 4px

content-box:
реальна ширина = 200 + 40 + 8 = 248px

width: 200px
padding: 20px
border: 4px

border-box:
реальна ширина = 200px (padding і border всередині)
