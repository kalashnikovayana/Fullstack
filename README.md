# Fullstack

Цей репозиторій містить домашні завдання з курсу Fullstack.

## Структура репозиторію

- `home-work-1` — домашнє завдання 1
- `home-work-2` — домашнє завдання 2
- `home-work-3` — домашнє завдання 3
- `home-work-4` — домашнє завдання 4

## Технології

- HTML
- CSS
- Git
- GitHub
- GitHub Pages

## GitHub Pages

Опубліковані домашні завдання можна переглядати через GitHub Pages.
